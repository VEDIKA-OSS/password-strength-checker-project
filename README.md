# 🔐 Password Strength Checker

This is a Python-based GUI application that checks the strength of a password and suggests a stronger alternative.

## 🚀 Features
- Password strength analysis (Very Weak → Very Strong)
- Entropy calculation
- Real-time validation
- Strong password suggestion generator
- GUI using Tkinter

## 🛠️ Technologies Used
- Python
- Tkinter
- Regex

## ▶️ How to Run
1. Install Python
2. Run the file:
   python app.py
